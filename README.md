## Proyecto de prueba Easy Broker

Para poner la aplicación en marcha ejecute los siguientes comandos en una consola dentro del directorio donde se haya clonado el repositorio:

 `npm install`

 `npm runGetUp`

 `npm start`
 
 Para poder ejecutar los test:
 
 `npm run test-server`
 
 `npm run test`
